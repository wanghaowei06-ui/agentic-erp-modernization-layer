# Runbook

## Environment

Use the local Python virtual environment:

```bash
cd /home/changv/projects/Uipath
python3.11 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
chmod +x scripts/*.sh
```

## Start Services

```bash
./scripts/dev-start.sh
```

Ports:

- Mock Legacy ERP: `http://localhost:8001`
- reasoning-agent: `http://localhost:8002`
- generated-api-facade: `http://localhost:8003`
- validation-suite: `http://localhost:8004`

Stop services:

```bash
./scripts/dev-stop.sh
```

## Smoke Test

```bash
./scripts/smoke-test.sh
```

The smoke test clears `memory-data/`, exercises the Hard MVP support services, and verifies memory timeline, capabilities, and gaps.

## Release Candidate Check

Before recording or submitting:

```bash
make demo-check
```

This runs pytest, the smoke test, and evidence collection. The frozen baseline is documented in `docs/release-candidate.md` as `hard-mvp-automation-memory-rc1`.

## Clean Memory Data

```bash
rm -rf memory-data
```

Legacy compatibility artifacts remain under `memory/data/` and are regenerated by validation smoke paths.

## Verify Memory Timeline

After running smoke test:

```bash
curl -sS http://localhost:8004/memory/cases/CASE-001/timeline | .venv/bin/python -m json.tool
```

Expected event types include:

- `TRIAGE_COMPLETED`
- `RPA_WRITEBACK_COMPLETED`
- `VALIDATION_COMPLETED`
- `API_EXECUTION_COMPLETED`

## Verify Capability Registry

```bash
curl -sS http://localhost:8004/memory/capabilities | .venv/bin/python -m json.tool
```

Expected trusted capabilities:

- `cap_api_request_po_approval_v1`
- `cap_human_po_approval_v1`

## Verify Capability Gap

```bash
curl -sS http://localhost:8004/memory/gaps | .venv/bin/python -m json.tool
```

Expected event:

- `CAPABILITY_GAP_RECORDED`

## Troubleshooting

If Windows cannot open `localhost`, check the WSL IP:

```bash
hostname -I
```

Use `http://<WSL-IP>:8001` through `http://<WSL-IP>:8004` from Windows if localhost forwarding is unavailable.

If smoke test fails on connection refused, start services first:

```bash
./scripts/dev-start.sh
```

If memory timeline is empty, rerun:

```bash
./scripts/smoke-test.sh
```
